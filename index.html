Frequently Asked Questions

How to show exact timestamps on GitHub?

Simply install the GitHub Absolute Dates Chrome extension and it will automatically replace relative timestamps with absolute dates on all GitHub pages. No configuration needed!

Does it affect performance?

No, the extension is lightweight and has minimal performance impact. It only processes timestamps that are already visible on the page.

Is it safe and secure?

Yes, the extension is completely safe. It only modifies the display of timestamps on GitHub pages and doesn't collect or transmit any data.

Will it work on private repositories?

Yes, the extension works on all GitHub pages including private repositories, as long as you have access to view them.

Can I customize the date format?

Currently, the extension uses a standard format, but we're working on customization options for future releases.
